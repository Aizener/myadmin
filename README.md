### waht's it?
> 这是一个基于vue2+element-ui做的一个后台管理系统，想做的一共有：
1. vue+element-ui搭起页面；
2. 做好最基本的权限管理(或许要搭配express)；
3. 使用echarts来做可视化；
4. 使用i18n来做一下国际化；
5. 暂时以上这么多，能先做完就不错了...
